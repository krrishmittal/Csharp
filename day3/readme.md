# Day 3 - Arrays, Methods, and Classes

## Topics Covered

1. **String Arrays** - Creating and working with arrays of strings, taking user input
2. **2D Arrays** - Working with two-dimensional arrays using nested loops
3. **Jump Statements** - Using `break` to exit loops, `continue` to skip iterations, and `goto` to transfer control
4. **Array Sorting** - Using `Array.Sort()` to sort arrays in ascending order
5. **Binary Search** - Using `Array.BinarySearch()` to search for elements in sorted arrays
6. **Classes and Objects** - Understanding how classes define blueprints and objects are instances with allocated memory
7. **Class Members** - Working with fields, properties, and methods within classes
8. **Methods in C#** - Creating reusable blocks of code with parameters and return types
9. **Method Calling**:
   - **Direct Method Calling** - Invoking instance methods using object references
   - **Static Method Calling** - Invoking static methods using class names without instantiation
10. **Method Parameters**:
    - **Value Parameters** - Pass copies of arguments
    - **Reference Parameters** - Pass references using `ref` keyword
    - **Output Parameters** - Return multiple values using `out` keyword
