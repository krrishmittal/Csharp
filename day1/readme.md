# Day 1 - C# Basics

## Topics Covered

1. **Printing Statements** - Using `Console.WriteLine()` to output text to the console
2. **User Input** - Taking input from users with `Console.ReadLine()` and displaying it
3. **Integer Calculations** - Creating custom classes and methods for arithmetic operations (addition, subtraction, multiplication, division)
4. **Control Statements** - Using `if-else` conditions for decision-making logic
5. **Loops** - Implementing `for` loops for iteration
6. **Classes in C#** - Defining and using classes with methods
7. **Variables** - Working with different data types (int, string, bool, double)
8. **Arrays** - Creating and iterating through arrays using loops
