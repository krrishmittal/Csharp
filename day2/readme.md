# Day 2 - C# Intermediate Concepts

## Topics Covered

1. **Classes and Objects** - Creating classes and instantiating objects in C#
2. **Inheritance** - Using class inheritance to extend functionality from parent classes
3. **If-Else Statements** - Conditional logic for decision-making
4. **Out Parameters** - Using `out` keyword to return multiple values from a method
5. **Arrays** - Creating and working with arrays, including array input from users
6. **Foreach Loops** - Iterating through array elements using `foreach` loop
7. **Params Keyword** - Using `params` to accept a variable number of arguments in a method
8. **Method Return Values** - Returning and handling multiple values from methods
